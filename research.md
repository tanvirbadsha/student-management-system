# Actors

Staff:

- create students
- manage fees
- create assessments
- enter grades
- publish results

Student:

- submit assessments
- view own results
- maybe see fee balance.

# Core Entities

- Student (student info)
- Programme (academic programme)
- Fee (assigned fee)
- Payment (payment transaction)
- Assessment (assignment/exam)
- Submission (uploaded work)
- Result (marks/ grade)
- userRole (student/staff)
